usb切换到wifi

  1. 手机通过usb线连接上电脑
  2. adb tcpip 5555
  3. 可以断开usb连接
  4. adb connect 192.x.x.x 修改为android设备的ip
  5. 通过adb进行操作
  6. adb disconnect 断开连接
