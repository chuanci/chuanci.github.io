AndroidUI之Toast

发表于 2016-04-02 | | 本文总阅读量 次

一个可视控件,用于向用户显示一些提示.

注意

  1. 只需要一个Context,就可以使用.所以在Service中也能用.
  2. 要在主线程中调用.

方法

  • Toast.makeText(context,text,duration);//返回一个toast
  • toast.setDuration(duration);//设置持续时间
  • toast.setGravity(gravity,xOffset,yOffset);//设置显示位置
  • toast.setText(s);//设置显示内容
  • toast.show;//显示

几种Toast

默认Toast

Toast.makeText(MainActivity.this, getString(R.string.toast1), Toast.LENGTH_SHORT).show();

带图片的Toast

Toast toast = Toast.makeText(MainActivity.this, getString(R.string.toast2), Toast.LENGTH_SHORT);
toast.setGravity(Gravity.CENTER, 0, 0);
LinearLayout layout = (LinearLayout) toast.getView();
ImageView imageView = new ImageView(getApplicationContext());
imageView.setImageResource(R.mipmap.ic_launcher);
layout.addView(imageView, 0);
toast.show();

改变位置的Toast

Toast toast = Toast.makeText(MainActivity.this, getString(R.string.toast3), Toast.LENGTH_SHORT);
toast.setGravity(Gravity.TOP, 0, 0);
toast.show();

自定义的Toast

Toast toast = new Toast(getApplicationContext());
View view = View.inflate(getApplicationContext(), R.layout.toast_layout, null);
toast.setView(view);
toast.setDuration(Toast.LENGTH_SHORT);
toast.show();
