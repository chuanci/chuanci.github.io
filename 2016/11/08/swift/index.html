swift

发表于 2016-11-08

Swift，是苹果于2014年WWDC（苹果开发者大会）发布的新开发语言，可与Objective-C*共同运行于Mac OS和iOS平台，用于搭建基于苹果平台的应用程序。

2015年12月4日，苹果公司宣布其Swift编程语言现在开放源代码。

xcode

苹果发布的语言，自然工具就要选用xcode

常量与变量

常量 通过let声明 

let NUM=1

变量 通过var声明

var i=1

虽然用let和var来声明，但是swift是一个强类型语言，所以我们也可以声明类型

let s:String = "hello world"

swift中，变量名称可以使用unicode中的任意字符，即可以使用中文

常用数据类型

整型(默认Int)

Int

最大值是根据计算机来定的，32位计算机就是32位，64位计算机就是64位

Int.max//最大值 9223372036854775807
Int.min//最小值 -9223372036854775808

如果溢出会在编译时报错

UInt

UInt.max//最大值 18446744073709551615
UInt.min//最小值 0

角标

print(point2.0)
print(point2.1)

命名元祖分量1

print(point3.y)

选择结构

if else

switch

与其他语言不一样，不需要每个case都写一个break

var rating="a"

switch rating{
    case "a","A"://可以匹配多个
        print("AA")
    case "b","B":
        print("BB")
    case "c","C":
        print("CC")
default:
    （）//空语句
}

if case

在switch中

let x=10

switch x {
case 1...20:
    print(x)
default:
    ()
}

用if case

let x=10

if case 1...20 = x{
    print(x)
}

where

用于限制case,在swift3中，可省略where，用,代替

guard

在函数中，我们会进行边界判定，苹果推荐使用guard来进行边界判定

func fun( name : String , age : Int){
    guard  age<=20 else {
        print("age  为 \(age) >20")
        return
    }

    guard age>=10 else {
        print("age 为 \(age) <10")
        return
    }

    print("fun")

    }
